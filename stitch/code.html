01. Sobre Mí

Mi viaje en el desarrollo web comenzó con la curiosidad de entender cómo funcionaba el internet. Lo que empezó como un pasatiempo experimentando con HTML y CSS se convirtió en una carrera profesional donde la ingeniería de precisión se encuentra con el diseño elegante.

Mi enfoque principal hoy en día es crear aplicaciones robustas y escalables. Me apasiona el código limpio, el rendimiento y la creación de interfaces que no solo se vean bien, sino que se sientan naturales al usarlas.
